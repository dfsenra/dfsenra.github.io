Pipeline de Coleta de Dados Financeiros (YFinance)

Desenvolvimento de um pipeline de dados em Python para coleta automatizada de preços históricos de ativos financeiros.

A solução implementa um ETL robusto com controle de rate limit para evitar bloqueios de IP, filtros estatísticos para diferenciar cotações de proventos, além de logs auditáveis, alertas por e-mail e execução automatizada.

O projeto prioriza reprodutibilidade, clareza de código e criação de uma base sólida para análises e modelos futuros.

Development of a Python-based data pipeline for automated collection of historical financial asset prices.

The solution implements a robust ETL with rate-limit control to avoid IP blocking, statistical filters to distinguish prices from dividends, as well as auditable logs, email alerts and automated execution.

The project prioritizes reproducibility, code clarity and the creation of a solid foundation for future analyses and models.

Stack: Python, Pandas, YFinance
